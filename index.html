Voice



Each note (or rest) lasts one beat (60 ÷ BPM seconds). For example, at 120 BPM each beat lasts 0.5 seconds. The natural octave sets the base tone for "Do" (e.g. C4). Allowed natural octaves: between A1 and D6.

Octave markers:
• Single high: "/" or "!" (one octave higher)
• Double high: "//" or "!!" (highest octave; available only if Range is Higher)
• Single low: "\" or "¡" (one octave lower)
• Double low: "\\" or "¡¡" (lowest octave; available only if Range is Lower)
• "~" resets to natural.
If no marker is present, the note is assumed to be in the natural octave.

Play a Sequence

Enter a sequence that may include note symbols (D, d, R, r, M, F, f, S, s, L, l, T), octave markers as defined above, and spaces (a space is a one‐beat rest).
For example:
– If natural octave is C4 and Range is Higher: "//D~D\D" plays: C6, C4, C3.
– If natural octave is C4 and Range is Lower: "\D" plays C2.
